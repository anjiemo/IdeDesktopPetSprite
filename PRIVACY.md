# Privacy Policy — IDE Desktop Pet Sprite

Last updated: 2026-06-30

## Summary

This plugin does **not** collect, store, or transmit personal data to the developer (anjiemo).

## Data stored locally

The plugin stores preferences on your machine only — for example pet position, size, selected character, and per-project settings — using IntelliJ Platform persistent storage. This data never leaves your device unless you back up your IDE settings yourself.

## Network access (optional)

If you use the **Petdex** online character library, the plugin downloads public sprite assets from:

- `https://petdex.dev` (manifest)
- `https://assets.petdex.dev` (sprite files)

These requests are initiated only when you open the character picker, refresh the catalog, or download a character. Standard HTTP/TLS metadata (such as IP address) may be processed by Petdex servers under their own policies. The plugin does **not** send your source code, project names, file paths, or personal identifiers to Petdex or to the plugin author.

If you do not use Petdex, the plugin does not require network access for its core features (built-in character and local sprite upload).

## Third-party services

Petdex is operated independently. Its privacy practices are governed by [petdex.dev](https://petdex.dev) and the respective asset authors. Characters from Petdex are not bundled with this plugin.

# 隐私政策 — IDE Desktop Pet Sprite

最后更新：2026-06-30

## 概要

本插件**不会**向开发者（anjiemo）收集、存储或传输任何个人数据。

## 本地存储

插件仅在您的计算机上保存偏好设置（例如宠物位置、尺寸、所选形象、按项目配置等），使用 IntelliJ 平台的持久化存储。除非您自行备份 IDE 设置，这些数据不会离开您的设备。

## 网络访问（可选）

若您使用 **Petdex** 在线形象库，插件会从以下地址下载公开的精灵图资源：

- `https://petdex.dev`（形象清单）
- `https://assets.petdex.dev`（精灵图文件）

仅在您打开形象选择器、刷新形象库或下载形象时才会发起上述请求。Petdex 服务器可能按其自身政策处理标准 HTTP/TLS 元数据（如 IP 地址）。插件**不会**向 Petdex 或插件作者发送您的源代码、项目名称、文件路径或个人标识信息。

若您不使用 Petdex，插件核心功能（内置形象、本地上传）无需联网。

## 第三方服务

Petdex 由独立第三方运营，其隐私实践受 [petdex.dev](https://petdex.dev) 及各素材作者条款约束。Petdex 形象不随本插件分发。
